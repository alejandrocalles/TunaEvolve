# TunaEvolve

ShinkaEvolve with Reinforcement Learning Fine-Tuning


### Quickstart

```bash
# Create conda environment (needed for sagemath)
conda create env -f environment.yaml

# `uv` will install the rest of the packages into the previous environment
uv sync --active

# Run the programs using `uv run`
uv run shinka_launch [args...]
uv run shinka_visualize [args...]
```
